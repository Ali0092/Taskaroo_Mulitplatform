# Changelog

## [1.0.0] - 2025-12-30

### Added

- Initial release of Taskaroo - Cross-platform task management application
- Task creation and management functionality
- Priority-based task organization (Urgent, High, Medium, Low)
- Color-coded priority indicators for visual task management
- Calendar-based task viewing with horizontal scrollable calendar widget
- Cross-platform support for Android (SDK 24+) and iOS (14.1+)
- Local SQLDelight database for task persistence and offline-first functionality
- User preferences and settings management
- Onboarding flow for new users with introduction screens
- Basic user information collection screen
- Edit task functionality with pre-filled forms
- Delete task functionality with confirmation
- Material Design 3 interface with custom theming
- Reusable UI components library
- Platform-specific implementations using Kotlin's expect/actual pattern
- Database operations layer with type-safe queries
- Date and time utilities for task scheduling
- Navigation system using Voyager library

### Technical Stack

- Kotlin 2.2.21
- Compose Multiplatform 1.9.3
- SQLDelight 2.0.1 for database management
- Voyager 1.1.0-beta03 for navigation
- kotlinx-datetime 0.5.0 for date/time handling
- Material Icons Extended for comprehensive icon library
